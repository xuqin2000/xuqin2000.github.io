运算符总结

总结运算符

一、基础运算符（+-*/%）

注意：一旦运算中有不同类型的数据，那么结果是数据类型范围大的那种
四则运算中的加号：
1.对数值来说就是加法

2.对字符char类型来说，在计算之前，char会提升为int再计算，对照关系表ASCII、unicode

3.对于字符串Sring（首字母大写，并不是关键字）来说，加号代表字符串连接操作。
即任何数据类型和字符串进行连接的时候，结果都会变成字符串

string + int-->string
优先级：
string + int+ int>string + int>string

二、自增自减运算符

自增运算符：++
自减运算符：–
让一个变量涨一个数字1，或者减一个数字1

写在变量名称之前，或者写在变量名称之后，如++num，num++
1.单独使用的时候，前后++没区别
2.混合使用有区别
a.如果是前++，那么变量立刻+1，拿着结果进行使用（先加后用）
b.若果是后++ ，先使用变量本来的数值，再让变量+1（先用后加）
tips: 只有变量才可以用自增自减运算

三、赋值运算符

分为基本赋值运算符、复合赋值运算符
+= a+=1 相当于 a = a + 1

-= b-=2 相当于 b = b - 2

= c*=3 相当于 c = c * 3

/= d/= 4 相当于 d = d / 4

%= e%=5 相当于 e = e % 5

1.只有变量才能使用赋值运算符号，常量不能赋值
2.复合赋值运算符隐含了一个强制类型转换

四、比较运算符

==，<,>,<=,>=,!=

1.比较运算符的结果一定是一个Boolean值，即true或者false

2.进行多次判断，不能连着写

1<x<3（不允许）

五、逻辑运算符

与（并且）&& 两者为真才为真，一假就为假（true）

或（或者） | | 一真则为真，全假才是假（false）

非（取反） ！ 本来是true，变成false

与“&&，”或“”| |，具有短路效果：如果根据左边已经可以判断出最终结果，那么右边的代码不再执行，从而节省一定的性能 （false&&…左边已经能够判断出来true，所以后面不用计算）

注意：1.逻辑运算符只能为Boolean值

2.与、或需要左右各自一个Boolean值，但是非只要有唯一一个Boolean值即可

3.与、或两种运算符，如果有多个条件，可以连续写

两个：条件A&&条件B

多个：条件A&&条件B&&条件C

tip： 对于1<x<3,可以拆分成两个部分与运算符连接起来

int x=2；

1<x&&x<3；

六、多元运算符

一元运算符：！，++，–等

三元运算符：需要三个数据才可以进行操作的字符

数据类型 变量名称 = 条件判断 ？ 表达式A：表达式B

流程：

首先判断条件是否成立：

如果成立为true，那么将表达式A的值赋给左侧的变量;

如果不成立则为false，将表达式B的值赋给左侧的变量；

二者选其一。

注意：

1.必须同时保证表达式A和B都符合左侧数据类型的要求

2.三元运算符的结果必须被使用

七、位运算符：

补码的负数表示：例如-1:11111111 11111111 这里首位为1表示负数，直接算后面八位，即11111111。第一位为负数且表示为-128，后面剩下的七位则相加并且是正数，这样-128+127即为-1.

位运算符有七个，并且并不是运算数值，而是在二进制上进行运算。即&，|，~，^,<<,>>,>>>

位与运算符&：相同数保留，其余为0

位或运算符|：有1就保留

位异或运算符^：有相同就为0，不相同为1

位取反运算符~：取反，即0变成1,1变成0

左移、右移运算符：

<< 除以2

>> 乘以2
